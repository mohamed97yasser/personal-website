# 🌐 Mohamed Yasser - Personal Portfolio  

This is my **personal portfolio website**, built to showcase my skills, projects, and experience as a **Software Engineer** and **Front-End Web Developer**.  

The website includes information about me, my skills, featured projects, client testimonials, and contact details. It is fully **responsive**, **theme-switchable (dark/light)**, and styled with **modern CSS (SCSS)**.

---

## 🚀 Features  

- 🎥 Background looping video effect.  
- 📱 Fully responsive design (mobile, tablet, desktop).  
- 🎨 Light/Dark theme toggle.  
- 📑 Sections:  
  - **Bio/About Me**  
  - **Skills**  
  - **Media Showcase**  
  - **Featured Projects**  
  - **Clients Testimonials**  
  - **Contact Form**  
- 🖼️ Smooth animations & hover effects.  
- ⚡ Built with **HTML5 + SCSS + CSS Grid & Flexbox**.  

---

## 🛠️ Technologies Used  

- **HTML5**  
- **CSS3 / SCSS**  
- **Responsive Design** (Media Queries, CSS Grid, Flexbox)  
- **JavaScript (basic interactions)**
